# Runa Finance

React Native приложение на Expo.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Затем выберите платформу:
- `i` для iOS симулятора
- `a` для Android эмулятора
- `w` для веб-браузера

## Структура проекта

```
src/
  components/     # React компоненты
    LoadingView.tsx
```
